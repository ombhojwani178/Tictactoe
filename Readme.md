# Tic Tac Toe Game

## Overview
This project is a simple **Tic Tac Toe game** implemented using **HTML, CSS, and JavaScript**.  
It allows two players to play the game on a 3×3 grid following standard Tic Tac Toe rules.

## Objective
To build a basic interactive web-based game using front-end technologies.

## Features
- Two-player gameplay (X and O)
- Win and draw detection
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open `tictac.html` in a web browser.
3. Play the game.

## Files
- `tictac.html` – Structure of the game
- `style.css` – Styling of the game board
- `script.js` – Game logic
